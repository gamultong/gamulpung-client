## Contribute

### How to test 
1. Setting Environments
```
node >= 20.10
npm >= 10.9
```
2. Install Dependencies.
```bash
# Because React 19 does not support previous dependencies before version 19.
npm install --force 
```

3. Run Development Server.
```bash
npm run dev
```

4. Test Linting Codes Before Building.
```bash
npm run lint
```
